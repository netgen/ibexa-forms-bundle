Integration of Symfony's Form component with Ibexa Platform Public API

Currently implemented forms:

| Entity / Form             | Create | Update | Information collection
| ------------- | --- | --- | ---
| Content      | yes | yes | yes
| User      | yes | yes | no

Currently supported FieldTypes:

| FieldType    | Supported
|--------------| ---
| Author       | no
| BinaryFile   | yes
| Checkbox     | yes
| Country      | yes
| Date         | yes
| DateAndTime  | yes
| EmailAddress | yes
| Float        | yes
| Integer      | yes
| Image        | yes
| ISBN         | yes
| Keyword      | no
| Media        | no
| MapLocation  | yes
| Page         | no
| Rating       | no
| Relation     | yes
| RelationList | yes
| RichText     | no
| Selection    | yes
| TextBlock    | yes
| TextLine     | yes
| Time         | yes
| Url          | yes

| RichText     | no
